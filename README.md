Interactive Pool Table Scene - WebGL Graphics Project

An interactive 3D billiards scene built with pure native WebGL, featuring realistic animations, dynamic lighting, and immersive interaction. This is a dependency-free educational project for COMP4422 Computer Graphics.

Setup Instructions：
Prerequisites：
Modern web browser (Chrome 90+, Firefox 88+, Edge 90+)
No additional software or libraries required
Visual Studio Code

Launch the project:
Download the project files into a single folder
Open the folder in VS Code
Right-click index.html in VS Code
Select "Open with Live Server"

Interactive operations:
Mouse Drag - Rotate view to see from different angles
Spacebar - Trigger shot animation (cue hits → white ball → black ball pockets)
T Key - Switch to top-down view (looking vertically down)
A Key - Return to initial view (front view of pool table)

Install "Live Server" extension (if not already installed):
Open VS Code Extensions panel (Ctrl+Shift+X)
Search for "Live Server" by Ritwick Dey
Click Install
